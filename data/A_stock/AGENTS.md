# data/A_stock/

## Purpose
A-share (Chinese stock market) historical and real-time data storage.

## For AI Agents
Contains:
- Historical OHLCV data for A-share stocks
- Real-time market data feeds
- Stock listings and symbols
- Market indices data (Shanghai, Shenzhen)
- Corporate action data

## Dependencies
- Populated by `../../agent/base_agent_astock/`
- Processed by tools in `../../tools/`
- Configured by `../../configs/`
