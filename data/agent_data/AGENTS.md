# data/agent_data/

## Purpose
Trading results and performance data organized by AI model. Each AI agent has its own subdirectory for storing trades, P&L, and analytics.

## Subdirectories
- `claude-3.7-sonnet/` - Claude agent trading data
- `deepseek-chat-v3.1/` - DeepSeek agent trading data
- `gemini-2.5-flash/` - Gemini agent trading data
- `gpt-5/` - GPT-5 agent trading data
- `MiniMax-M2/` - MiniMax agent trading data
- `qwen3-max/` - Qwen agent trading data

## For AI Agents
Each subdirectory contains:
- Trade execution history
- Profit/loss records
- Performance metrics
- Position data
- Decision logs

## Dependencies
- Generated by all agents in `../../agent/`
- Analyzed by tools in `../../tools/`
- Visualized by `../../tools/plot_metrics.py`
