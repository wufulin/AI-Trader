# data/agent_data_astock_hour/

## Purpose
Hourly trading data for A-share market. Separate storage for granular time-series analysis of A-share trading performance.

## For AI Agents
Contains:
- Hourly trade executions
- Intraday performance metrics
- Hourly P&L tracking
- Intraday position changes
- Time-of-day analysis data

## Dependencies
- Generated by `../../agent/base_agent_astock/`
- Analyzed by tools in `../../tools/`
- Complements daily data in `../agent_data/`
