# data/crypto/

## Purpose
Cryptocurrency market data storage across multiple exchanges and trading pairs.

## For AI Agents
Contains:
- Historical OHLCV data for crypto assets
- Real-time price feeds
- Order book data
- Multiple exchange data (Binance, Coinbase, etc.)
- Trading pair information
- On-chain metrics (if applicable)

## Dependencies
- Populated by `../../agent/base_agent_crypto/`
- Processed by tools in `../../tools/`
- Configured by `../../configs/`
